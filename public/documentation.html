Documentation

This guide explains how to use our API server. It helps your apps connect with our services through easy-to-use endpoints. See below for endpoint details, parameters, and responses.

Method Route Description Request Body Response
POST /users Create a user JSON Object with user data
Example:
{"email": "user@example.com", "username": "newuser", "password": "password123"}		 Success: "User created!"
Error: "This email already exists"
PUT /users/:email Update user's username JSON Object with updated data
Example:
{"username": "updatedusername"}		 Success: "User updated!"
Error: "This email does not exist"
POST /users/:email/:movieName Favorite a movie for a user Not applicable Success: "Movie 'movieName' favorited!"
Error: "This email or movie does not exist"
DELETE /users/:email/:movieName Remove a movie from user's favorites Not applicable Success: "Movie 'movieName' removed from favorite!"
Error: "This email or movie does not exist"
DELETE /users Delete a user JSON Object with user email
Example:
{"email": "user@example.com"}		 Success: "User removed!"
Error: "This email was not found"
GET /movies Get all movies Not applicable Array of movie objects
GET /movies/:name Get a single movie by title Not applicable Single movie object
GET /movies/genre/:name Get movies by genre name Not applicable Array of movie objects
GET /movies/directors/:name Get movies by director name Not applicable Array of movie objects
